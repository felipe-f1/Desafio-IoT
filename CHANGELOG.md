# Changelog

## 1.0.0

- Implementa sensor virtual Python com média móvel, outliers e quedas de conexão.
- Adiciona broker MQTT Mosquitto.
- Implementa backend FastAPI com ingestão MQTT, WebSocket, JWT e cálculo de consumo em kWh.
- Adiciona TimescaleDB com hypertable e índice por timestamp.
- Implementa dashboard Next.js com gráfico em tempo real e cálculo de custo estimado.
- Adiciona Nginx como proxy reverso.
- Documenta execução, arquitetura e roteiro de demonstração.
